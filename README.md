# paathshala
A Learning Management System

#Guide to run the application

#Start to run backend server

step 1 : clone this repo
step 2 : run command " git pull origin feature-ai "
step 3 : create .env file and fill essential credentials in application.properties
step 4 : run command " docker compose up --build "

#Start to run frontend server

step 1 : run command " git clone https://github.com/Anish5703/spring-boot-react-starter.git "
step 2 : run command " npm run dev "
step 3 : access web app on url " http://localhost:8080"
